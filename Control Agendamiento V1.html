⚙️ Configurar Google Sheets
Sigue estos pasos para conectar el dashboard con Google Sheets:
  1. Abre Google Sheets y crea un nuevo archivo llamado LIMMAN_DB
  2. Ve a Extensiones → Apps Script
  3. Borra el código existente y pega el código del archivo AppScript.gs que descargaste junto con este dashboard
  4. Haz clic en Implementar → Nueva implementación → Aplicación web
  5. En "Ejecutar como" selecciona Yo. En "Quién tiene acceso" selecciona Cualquier persona
  6. Copia la URL generada y pégala abajo 👇
